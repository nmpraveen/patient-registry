V7: Patient Row Redesigns

Module-specific patient row treatments. Each module gets a row design tailored to its purpose: Today rows surface call status, Overdue rows show days-overdue urgency badges, Upcoming rows feature task chips, Recently Added uses relative time markers, and Awaiting Reports highlights report types.

  • Initials avatars and category color-coding for visual anchoring
  • Diagnosis, sex/age, and flags visible without expanding
  • Days-overdue urgency scale (mild to critical) for overdue module
  • Readable call-status badges replacing emoji indicators
  • Report-type color tags for awaiting reports module
