Design Philosophy

Context-Aware Rows
Each module shows different data because each serves a different purpose. Today rows prioritize call status and tasks. Overdue rows emphasize urgency. Upcoming rows focus on scheduling.
Visual Hierarchy
Patient names get avatars with initials for scanability. Key metadata surfaces inline without expanding. Color-coded urgency indicators replace emoji-only status.
Breathing Room
More vertical padding, subtle separators, and micro-detail lines give each row more personality without adding clutter. Rows feel alive, not like spreadsheet cells.
