MEDTRACK UI Design Variations

Three modern design directions for the patient registry system.
Each variation includes Login, Dashboard, Case List, and Case Detail pages.

MEDTRACK
V1: Glass Morphism

Frosted glass panels over vibrant gradients. Bold, futuristic aesthetic with translucent layers and ambient light effects.

  • Backdrop-filter blur effects on all panels
  • Gradient backgrounds with floating orbs
  • Top navbar with integrated search
  • Cyan accent colors on dark gradient

V2: Dark Clinical

Professional dark theme with a permanent sidebar. Data-dense layout using monospace typography for clinical precision.

  • Fixed sidebar navigation with user profile

V3: Warm Minimal

Clean, calming design with warm earth tones. Generous white space and soft borders create a friendly, approachable feel.
