Type scale → MaterialTheme

One named scale (display/title/card-name/body/sub/eyebrow/status) replaces ad-hoc .copy(fontSize=…) calls. Eyebrows gain size + tracking + the faint colour; headings get tight tracking; body floors at 13sp.

before · eyebrow 11.5 / muted / no trackafter · eyebrow 12.5 / 800 / faint / +0.6
2

FAB gradient = all-blue

The hardcoded #4F46E5 indigo literal is removed. The FAB becomes the 3-stop blue gradient Primary → PrimaryDark → PrimaryDeep with the shadow-fab token — so it reads as one brand blue, not blue-to-violet.

before · #2563EB → #4F46E5after · #2563EB → #1D4ED8 → #1E3A8A
3

Elevation & radius tokens

List cards get the subtle shadow-row (they're flat today); bucket chips snap to the full-pill radius token; the base card radius normalises to 16. Spacing/radius/elevation now come from tokens, not magic numbers.

before · flat · chip r18after · shadow-row · chip r999
Type scale — the clearest part of the change
Today the theme only swaps the font to Manrope and each screen sets its own sizes. P0 encodes this single scale once; the per-call overrides go away.
